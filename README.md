  - use new DB 

### Installation

DB requires [PHP 5.6]

Install the dependencies and devDependencies and start the server.

```sh
$ git clone https://github.com/gotzillas/classmysqli
$ echo <?php incude('DB.php'); $DB = new DB(); $DB->query("select * from table_name"); ?>
$ php -S 0.0.0.0:8080
```
